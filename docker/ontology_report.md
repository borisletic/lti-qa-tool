  - External application or service integrated into a Learning Management System

  - Learning Management System that consumes LTI tools (e.g., Canvas, Moodle)
- **LTI Provider**
  - Service that provides LTI tool functionality according to IMS standards
- **LTI Standard**
  - Version of IMS Learning Tools Interoperability standard
- **Learning Resource**
  - Educational material used in course (documents, videos, slides, etc.)
- **Question**
  - Student question about course material
- **Question Answering Tool**
  - Specialized LMS tool that provides automated question answering based on course materials
- **Student**
  - User enrolled in courses as a learner
- **Tool Launch**
  - Instance of LTI tool being launched by a user in specific course context
- **User**
  - Person using the LMS and integrated tools
